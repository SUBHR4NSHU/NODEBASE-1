# Abstract

Modern software ecosystems demand seamless integration between disparate services, AI models, and business workflows. However, the complexity of wiring together HTTP endpoints, event triggers, third-party APIs, and large language models typically requires significant programming expertise, making automation inaccessible to a broad range of users. Existing platforms such as Zapier, Make.com, and n8n offer workflow automation, but they either lack native support for cutting-edge AI models or impose proprietary, low-extensibility environments that hinder custom integration.

**NODEBASE** is a visual, node-based workflow automation platform designed to bridge this gap. The objective of this project is to provide an intuitive, drag-and-drop interface through which users can design, deploy, and monitor complex automation pipelines without writing boilerplate integration code. NODEBASE natively integrates leading AI providers — OpenAI, Google Gemini, and Anthropic Claude — as first-class workflow nodes, alongside event triggers (webhooks, Google Forms, Stripe) and communication services (Slack, Discord).

The system was developed using a modern full-stack architecture: **Next.js 15** (App Router) with **React 19** and **Tailwind CSS / Shadcn UI** on the frontend; **tRPC** for type-safe client–server communication; **PostgreSQL** managed via **Prisma ORM** for persistent storage; and **Inngest** as a durable, event-driven background execution engine. The visual canvas is powered by **`@xyflow/react`** (React Flow), enabling users to model workflows as **Directed Acyclic Graphs (DAGs)** composed of typed nodes and directed connection edges. Authentication and billing are handled through **Better Auth** and the **Polar SDK** respectively.

Key outcomes of the project include a fully operational workflow editor capable of constructing and persisting multi-node pipelines; a reliable execution engine that traverses the node DAG in dependency order via Inngest event steps, tracking each run with a status of `RUNNING`, `SUCCESS`, or `FAILED`; a secure credential vault for per-user API key management; and a real-time execution log viewer. The platform successfully demonstrates end-to-end automation — from an external trigger (e.g., a Stripe payment event or a Google Form submission) through AI processing to a downstream notification in Slack or Discord.

In conclusion, NODEBASE establishes a scalable, extensible foundation for AI-augmented workflow automation. By abstracting infrastructure concerns behind a visual interface and leveraging durable execution semantics, the platform enables rapid automation of complex business processes. Future scope includes support for conditional branching nodes, loop constructs, expanded third-party integrations (Notion, Gmail, HubSpot), and team collaboration features such as shared workflow templates and role-based access control.
